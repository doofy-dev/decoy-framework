# decoy-framework
Simple PHP framework

##Requirements
-doctrine/orm : 2.4.*
-symfony/yaml : 2.*
##Installing
###using composer.json:
	"require": {
		"teeebor/decoy-framework":"dev-master"
	}
###using command line:
	composer require teeebor/decoy-framework
